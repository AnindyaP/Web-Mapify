# PROJECT MAPIFY 
Mapify adalah sebuah platform berbasis Sistem Informasi Geografis (SIG) yang digunakan untuk menampilkan dan memetakan lokasi secara interaktif melalui peta digital. Platform ini memungkinkan pengguna untuk mencari, menelusuri, dan mengetahui informasi detail tentang suatu tempat dengan mudah. Dalam konteks UMKM, Mapify berfungsi sebagai media promosi digital yang membantu usaha kecil agar lebih mudah ditemukan oleh masyarakat. Dengan fitur seperti pencarian lokasi, rute, dan tampilan profil usaha, Mapify mendukung peningkatan visibilitas serta aksesibilitas UMKM di wilayah Gunungpati.

HTML, interact js, css, dan framework bootstrap.
